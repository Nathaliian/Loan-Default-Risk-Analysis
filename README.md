# 📊 Loan Default Risk Analysis & Prediction

## 📌 About
This project analyzes borrower demographics, financial behavior, and loan characteristics to identify the key factors contributing to loan defaults. The analysis focuses on understanding rising default rates in personal lending and provides data-driven insights to improve risk assessment and the loan underwriting process.

---

## 🎯 Objectives
- Identify major risk factors behind loan defaults
- Analyze borrower behavior using statistical techniques
- Perform segmentation analysis across different customer groups
- Evaluate relationships between financial variables and default risk
- Generate business insights through visualization and reporting

---

## 📂 Project Workflow

### 1. Data Cleaning & Preprocessing
- Handled missing values
- Corrected data types
- Created binary target variable (`default_binary`)

### 2. Exploratory Data Analysis (EDA)
- Loan distribution analysis
- Borrower demographic analysis
- Default trend visualization
- Financial behavior analysis

### 3. Segment Analysis
Analyzed default rates based on:
- Credit Score
- Debt-to-Income Ratio (DTI)
- Employment Status
- Loan Purpose

### 4. Correlation & Statistical Testing
- Pearson Correlation Analysis
- Point-Biserial Correlation
- P-Value Significance Testing

---

## 📈 Key Findings
- Higher credit scores are associated with lower default risk
- Interest rates have a positive relationship with defaults
- Credit score and interest rate show strong negative correlation
- DTI has a weak but statistically significant relationship with defaults
- Loan amount showed little significance in predicting defaults
